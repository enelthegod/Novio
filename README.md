# 🏢 Novio — Job Board Platform

A modern job board where companies post vacancies and applicants apply.
Inspired by LinkedIn Jobs / Indeed.

## Tech Stack

**Backend**
- ASP.NET Core 8 Web API — Clean Architecture
- Entity Framework Core + PostgreSQL
- JWT Authentication with roles (Admin, Employer, Applicant)
- AutoMapper, FluentValidation

**Frontend**
- React 18 + TypeScript
- TailwindCSS, React Router, Axios

**DevOps**
- Docker Compose

## Project Structure

- `backend/` — ASP.NET Core API
- `frontend/` — React app
